🌟 项目愿景 (Vision)
车灵 AI 旨在打破汽车品牌间的“信息孤岛”。通过深度融合 RAG (检索增强生成) 技术与 LLM (大语言模型)，为用户提供一个跨品牌、跨语种、具备预测性交互能力的智能汽车技术顾问。

目前项目已成功接入 比亚迪 (Qin PLUS) 与 特斯拉 (Model Y)，未来将通过模块化设计，实现“分钟级”新车型接入，打造覆盖全行业的汽车数字大脑。

🛠️ 核心架构 (Core Tech Stack)
为了实现“六边形战士”般的表现，我们采用了以下技术栈：

感知层 (LLM): DeepSeek-V3 (负责意图识别、逻辑推理与动态建议生成)。
记忆层 (Vector DB): ChromaDB + Zhipu Embedding-3 (高效存储与检索百万级手册片段)。
交互层 (UI): Streamlit (打造类 Gemini 的流式思考与预测性 UI 体验)。
插件层 (Tools): 实时天气 API、汽车技术术语库。
✨ 核心特性 (Features)
🔍 跨品牌对比检索: 支持同时对比不同品牌的操控逻辑（如：特斯拉与比亚迪灯光开启方式的差异）。
🧠 类 Gemini 预测性交互: 实时分析对话语境，自动推测并推荐用户可能感兴趣的下三个问题。
🌐 全能知识融合: 结合“手册私有知识”与“大模型通用知识”，确保问不倒，答得准。
⚡ 极速响应架构: 采用多级缓存与 WebSocket 优化方案，告别初始化白屏。
🗺️ 远景规划 (Roadmap)
第一阶段：多车型矩阵 (进行中)
 比亚迪、特斯拉核心手册接入。
 接入小米 SU7、华为问界、理想等主流国产车型。
 支持多国语言手册自动翻译与汇总。
第二阶段：多模态感知 (即将到来)
 视觉诊断: 上传汽车仪表盘故障灯照片，AI 自动定位手册解决方案。
 语音交互: 接入语音识别，实现真正的驾驶场景免动手查询。
第三阶段：行业赋能
 接入实时维保数据，提供个性化用车建议。
 开放 API 接口，支持车企快速部署私有化手册 Agent。
